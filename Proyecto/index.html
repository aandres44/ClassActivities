Differences between Frontend and Backend

.

Frontend

  • Client-side of application or web page
  • Users can see and interact with it
  • Visual aspects of the application
  • Main programing languages: Html,css,javaScript

Backend

  • Server-side of the application or web page
  • Behind the scenes
  • Web server that links to database communication, to serve requests from Frontend
  • Brain of the website, never visible to user
  • Main programing languages: Ruby, Python, Java, .Net

.

End User

Is the identity or person that will interact with the application or web page, an analysis should be made regarding how the user will behave and what preferences he could have.

Client

The person, entity, company or whoever hired you to design and develop the application or web page.

.

CDN Architecture

CDN or content delivery network are servers strategically placed across the globe with the purpose of caching site content for faster delivery.
